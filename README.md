## Featured Projects

### 🔍 [NL-to-SQL Agent](https://github.com/venkatasandeepdangeti/agentasdataanalyst)
Ask a question in plain English, get the generated SQL, the result, and a one-sentence explanation — with a safety layer that guarantees the AI can only ever read data, never modify it.
**Live demo**: https://ftbkfugig5aod9sq2g2mhm.streamlit.app/

### 📦 [AdventureWorks Inventory & Restocking Agent](https://github.com/venkatasandeepdangeti/adventureworks-inventory-agent)
Same agent pattern, applied to real inventory/purchasing data from Microsoft's official AdventureWorks sample database — ask about restocking, vendor lead times, or warehouse stock levels.
